L15 Public Testnet

This testnet is ephemeral, meaning it is not meant to be fully stable and usable yet for more persistent test smart contracts. Assume the network could be reset at any time. The L16 Public Testnet is meant as a more stable persistent testnet before mainnet (Not yet running).

How to run an archive nodeâ€‹

  • Linux
  • MacOS
$ curl https://install.l15.lukso.network | bash
$ lukso start --node-name "REPLACE-WITH-NODE-NAME"
$ sudo curl https://install.l15.lukso.network | bash
$ sudo lukso start --node-name "REPLACE-WITH-NODE-NAME"

This starts your node as an archive node.

For all options and Windows please look at our LUKSO CLI documentation.

How to become a validatorâ€‹

caution

In case you are currently running a node, please run lukso stop (sudo lukso stop for MacOS) before continuing.

Step 1: Go to the LUKSO Launchpad and follow the steps to become a validator.

Step 2: You can check your validator status in the Vanguard Block Explorer. You can find your validator via its public key.
