LSP3UniversalProfile

deployâ€‹

lspFactory.LSP3UniversalProfile.deploy(
  profileDeploymentOptions,
  contractDeploymentOptions?);

Deploys and configures a Universal Profile to the blockchain. It will deploy the following contracts:

  • LSP0 ERC725 Account
  • LSP1 Universal Receiver Delegate
  • LSP6 Key Manager

Then, it will:

  • upload to IPFS and set the LSP3 Universal Profile metadata.
  • attach the Universal Receiver Delegate to the LSP0 ERC725 Account.
  • set the Key Manager as the owner of the LSP0 ERC725 Account.
  • give all permissions to the controllingAccounts.

Asynchronous version of deployReactive.

Parametersâ€‹

  1. profileDeploymentOptions - Object: The options used for deployment.
    • controllingAccounts - string[]: A list of accounts (public addresses) which will be granted all permissions on the newly created Universal Profile.
    • lsp3Profile? - Object: If set, the created Universal Profile will be populated with these values.
      • name - string: The name of the Universal Profile
      • description - string: The description of the Universal Profile
      • profileImage? - File | ImageBuffer | LSP3ProfileImage[]
      • backgroundImage? - File | ImageBuffer | LSP3ProfileImage[]
      • tags? - string[]
      • links? - {title: string, url: string}[]
  2. contractDeploymentOptions? - Object

Returnsâ€‹

Promise<Object>

Promise with object containing deployed contract details

Exampleâ€‹

await lspFactory.LSP3UniversalProfile.deploy({
  controllingAccounts: ['0xb74a88C43BCf691bd7A851f6603cb1868f6fc147'],
  lsp3Profile: {
    name: 'My Universal Profile',
    description: 'My cool Universal Profile',
    tags: ['Fashion', 'Design'],
    links: [{ title: 'My Website', url: 'www.my-website.com' }],
  },
});

deployReactiveâ€‹

lspFactory.LSP3UniversalProfile.deployReactive(
  profileDeploymentOptions,
  contractDeploymentOptions?);

Please check the asynchronous version.

Parametersâ€‹

Same as for the asynchronous version.

Returnsâ€‹

Observable<LSP3AccountDeploymentEvent | DeploymentEventTransaction>

RxJS observable which emits events as UniversalProfile contracts are deployed.

await lspFactory.LSP3UniversalProfile.deployReactive({
  controllingAccounts: ['0x9Fba07e245B415cC9580BD6c890a9fd7D22e20db'],

  next: (deploymentEvent) => {

uploadProfileDataâ€‹

LSP3UniversalProfile.uploadProfileData(profileData, uploadOptions?);

Uploads the LSP3Profile data to the desired endpoint. This can be an https URL either pointing to a public, centralized storage endpoint or an IPFS Node / Cluster.

Will upload and process passed images.

Parametersâ€‹

  1. profileData - Object
  2. uploadOptions? - Object

Returnsâ€‹

Promise<LSP3ProfileDataForEncoding>

Processed LSP3 data and upload URL.

Exampleâ€‹

await LSP3UniversalProfile.uploadProfileData({
  name: 'My Universal Profile',
  description: 'My cool Universal Profile',
