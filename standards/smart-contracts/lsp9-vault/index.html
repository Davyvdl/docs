LSP9Vault

The LSP9Vault contract is an implementation for the LSP9-Vault Standard.

This contract can be used as a vault that can hold assets and interact with other smart contracts, as it has all the functions that the LSP0ERC725Account contract have except isValidSignature function.

note

LSP9Vault contract also contains the methods from ERC165 :

  • supportsInterface (bytes4 interfaceId) public view returns (bool) :::

executeâ€‹

  function execute(
    uint256 operationType,
    address to,
    uint256 value,
    bytes memory data
  ) public payable returns (bytes memory result)

Executes a call on any other smart contracts, transfers value, or deploys a new smart contract.

The operationType can execute the following operations:

  • 0 for CALL
  • 1 for CREATE
  • 2 for CREATE2
  • 3 for STATICCALL
  • 4 for DELEGATECALL

Triggers the Executed event when a call is successfully executed using CALL/STATICCALL/DELEGATECALL operations.

Triggers the ContractCreated event when a smart contract is created using CREATE/CREATE2 operations.

It can only be called by the current owner of the contract. :::

Parameters:â€‹

NameTypeDescription
operationTypeuint256The operation to execute.
toaddressThe address to interact with. to will be unused if a contract is created (operation 1 & 2).
valueuint256The desired value to transfer.
databytesThe call data (ABI of the function to execute) , or the contract data to deploy.

Return Values:â€‹

NameTypeDescription
resultbytesThe returned data of the called function, or the address of the contract created (operation 1 & 2).

universalReceiverâ€‹

  function universalReceiver(
    bytes32 typeId,
    bytes memory data
  ) public returns (bytes memory result)

Forwards the call to the LSP1UniversalReceiverDelegateVault contract if the owner of the vault has set the LSP1UniversalReceiverDelegate Key to the address of the LSP1UniversalReceiverDelegateVault contract.

The LSP1UniversalReceiverDelegateVault contract should implement LSP1UniversalReceiverDelegate InterfaceId using ERC165.

Triggers the UniversalReceiver event when this function get executed successfully.

Parameters:â€‹

NameTypeDescription

databytesThe data received.

Return Values:â€‹

NameTypeDescription
